# EmployeeFormProject

A NetBeans HTML5/JS project for managing Employee records using JsonPowerDB (JPDB).

## Database Details
- **Database:** EMP-DB
- **Relation:** EmpData
- **Fields:** id, name, salary, hra, da, deduction

## How to Open in NetBeans
1. Open NetBeans IDE
2. Go to **File > Open Project**
3. Browse to and select the `EmployeeFormProject` folder
4. Click **Open Project**
5. Right-click `index.html` > **Run File**

## How to Use
- Enter an Employee ID and click **Search**
- If ID is **new** → fill form → click **Save**
- If ID **exists** → data loads automatically → edit → click **Change**
- Click **Reset** to clear the form at any time
