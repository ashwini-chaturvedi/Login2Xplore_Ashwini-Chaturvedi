#  Login2Xplore — Internship Project Repository
### Ashwini Chaturvedi · Round-2 Learning Assignments



This repository contains all hands-on assignments and project submissions completed as part of the **Round-2 recruitment process at Login2Xplore**. Each project demonstrates practical skills in front-end development and database integration using JsonPowerDB.

---

## Projects

### 1. [First Work](https://github.com/ashwini-chaturvedi/Login2Xplore_Ashwini-Chaturvedi/blob/main/FirstWork.html)
Introductory assignment exploring foundational HTML structure and JsonPowerDB API integration — the starting point of the internship journey.

**Tech:** HTML · JsonPowerDB

---

### 2. [Employee Form Project](https://github.com/ashwini-chaturvedi/Login2Xplore_Ashwini-Chaturvedi/tree/main/EmployeeFormProject)
A dynamic employee data entry form with full **CRUD operations** — insert, retrieve, update, and reset records — powered by JsonPowerDB as the backend database.

**Tech:** HTML · JavaScript · JsonPowerDB · REST API

---

### 3. [Student Enrollment Project](https://github.com/ashwini-chaturvedi/Login2Xplore_Ashwini-Chaturvedi/tree/main/StudentEnrollmentProject) (Final Assignment/Project)
A complete student enrollment management system featuring form validation, real-time data fetch by roll number, and save/update/reset functionality via JsonPowerDB.

**Tech:** HTML · JavaScript · JsonPowerDB · Form Validation
